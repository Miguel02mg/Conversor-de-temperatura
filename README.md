# Conversor-de-temperatura
Conversor de Temperatura
Descripción

Este proyecto consiste en una aplicación web desarrollada con HTML, CSS, Bootstrap y JavaScript que permite convertir una temperatura ingresada en grados Celsius a grados Fahrenheit y Kelvin.

El usuario introduce un valor numérico en un formulario y, al presionar el botón Convertir, la aplicación realiza los cálculos y muestra los resultados en pantalla.

Además, el programa valida que la entrada sea un número válido. Si el usuario introduce texto, deja el campo vacío o ingresa un valor incorrecto, se muestra un mensaje de error y se solicita un dato válido.

Objetivo

Aplicar conceptos fundamentales de JavaScript como:

-Variables.
-Funciones.
-Eventos.
-Condicionales.
-Validación de datos.
-Manipulación del DOM.
-Conversión de tipos de datos.

Tecnologías utilizadas:
-HTML5
-CSS3
-Bootstrap 5
-JavaScript

Funcionamiento:
El usuario ingresa una temperatura en grados Celsius.
El programa verifica que la entrada sea numérica.

Si la entrada es válida:

Convierte a Kelvin utilizando la fórmula:

K = °C + 273.15

Convierte a Fahrenheit utilizando la fórmula:

°F = (°C × 9/5) + 32

Los resultados se muestran en pantalla.
Si la entrada no es válida, se muestra un mensaje de error.
